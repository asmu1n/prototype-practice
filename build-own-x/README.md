# Build Your Own X

> Master programming by recreating your favorite technologies from scratch.

本目录用于存放 [codecrafters-io/BUILD-YOUR-OWN-X](https://github.com/codecrafters-io/BUILD-YOUR-OWN-X) 开源项目中各类技术的代码实践。

## 目的

通过从零实现经典技术，深入理解其底层原理，而非仅停留在 API 调用层面。

## 目录结构

```
build-own-x/
├── git/          # Build your own Git
└── README.md
```

## 已完成项目

| 项目 | 描述 |
|------|------|
| [git](./git/) | 从零实现简易 Git，理解版本控制原理 |

## 学习理念

每个项目的实现都遵循：

1. 理解核心概念与数据结构
2. 分析关键算法与流程
3. 从简化版本开始，逐步完善
4. 编写测试验证正确性
